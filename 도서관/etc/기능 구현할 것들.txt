유저모드와 관리자 모드 분리
로그인

유저모드
도서 찾기
대여 - 도서 찾기 후, 책ID 입력, 수량체크(반납 하면 수량 증가) -> 대여 성공 창(반납 기한 지난 책이 있다면 불가)
도서 대여 확인 - 빌린시간, 반납 기한(기한이 지났다면 빨간 표시)
도서 반납 - 책ID로 입력 후 반납
도서 반납 내역 - 빌린시간, 반납 시간
정보 수정 - PW, Name, Age, PhoneNumber, Address (뒤로가기 누르면 수정 안됨)
계정 삭제 - 빌린 도서가 있다면 불가 -> 탈퇴가 됐다면 로그인 창으로 돌아가기

매니저 모드
도서 찾기(유저모드와 동일)
도서 추가 - 입력 도움말 -> 추가 성공 창
도서 삭제 - 책 ID 입력 후 삭제
도서 수정 - 책 ID 입력 후 수정(도움말) -> 뒤로가기를 누르면 수정 X
회원 관리 - 유저 정보 수정, 유저 삭제(빌린 도서가 있다면 불 가능)
대여 내역 - 어떤 회원이 어떤 도서를 빌렸는지 확인하는 창
